# user-management-dashboard-vue

A frontend web application displaying a list of users fetched from a public API, allow searching and pagination, show user details in a modal, and allow the logged-in user to edit their own profile.

This is dashboard built with Vue.js 3, Pinia, and Vue Router, Vite.

## Features

- **Authentication**: Mock login with persistence using Pinia and localStorage.
- **User Directory**: Fetches 50 random users from the RandomUser API.
- **Refresh users button**
- **Search & Filter**: Real-time client-side search by name.
- **Pagination**: Efficiently browse large lists of users.
- **User Details**: Detailed modal view for each user.
- **Profile Management**: View and edit your profile information.

## Tech Stack

- **Framework**: Vue.js 3 (Composition API)
- **State Management**: Pinia
- **Routing**: Vue Router
- **Building Tool**: Vite
- **HTTP Client**: Axios
- **Styling**: Vanilla CSS (Custom Design System)

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```
